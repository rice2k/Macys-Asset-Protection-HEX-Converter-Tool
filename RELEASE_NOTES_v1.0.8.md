# Macy's AP China Grove Hex Utility v1.0.8

This release smooths out day-to-day workspace behavior and makes sample data simpler.

## Highlights

- Sample buttons now insert numeric-only examples.
- Input Queue and Unconvert Queue text is smaller for easier viewing.
- Single Lookup and FC/CN input fields are compact instead of full-width.
- Sidebar/footer status areas use fixed sizing so clicks do not shift the workspace/status layout.
- Results search row layout was cleaned up to avoid inconsistent movement.
