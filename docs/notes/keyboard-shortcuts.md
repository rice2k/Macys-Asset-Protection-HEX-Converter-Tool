# Keyboard Shortcuts And Mouse Actions

## Keyboard Shortcuts

| Shortcut | Action |
| --- | --- |
| `Ctrl+I` | Import files. |
| `Ctrl+R` | Convert Batch Converter input. |
| `Ctrl+E` | Export Excel workbook. |
| `Ctrl+Shift+E` | Export CSV report. |
| `Ctrl+P` | Export PDF report. |
| `Ctrl+F` | Jump to Results search. |
| `Ctrl+L` | Clear workspace. |

## Mouse Actions

| Action | Result |
| --- | --- |
| Mouse wheel over Input Queue | Scrolls queue. |
| Mouse wheel over result tables | Scrolls results. |
| Shift + mouse wheel | Horizontal scroll where supported. |
| Double-click converted result row | Copies FC,CN pair. |
| Right-click result row | Opens copy menu. |
| Hover table headers | Shows explanation for FC, CN, Status, and Notes / Details. |
| Drag supported files onto Input Queue | Imports files when drag/drop support is available. |
