Help — What this page does

AP Access Control Converter helps you convert access control IDs between: HEX (8 characters) ↔ Facility Code (FC) + Card Number (CN).

What it does

  • Hex ➜ FC/CN: Reads an 8‑character hex value and shows the Facility Code and Card Number.
  • FC/CN ➜ Hex: Builds the 8‑character hex value from Facility Code and Card Number.
  • Batch support: Paste multiple hex values (one per line) and convert them all at once.
  • Export: Downloads a formatted Excel file with borders and a header for printing or sharing.

How it works (simple)

The hex value is split into two halves:
First half (top 16 bits) = Facility Code
Second half (bottom 16 bits) = Card Number

How to use it

  1. Batch Convert: Paste your hex list (one per line) and click Convert All.
  2. Single Convert: Enter one hex value and click Convert Single.
  3. FC/CN to Hex: Enter numbers for Facility Code and Card Number, then click Convert To Hex.
  4. Export Excel (Formatted): Click to download a print‑ready spreadsheet.

Input rules

  • HEX must be exactly 8 characters (0–9, A–F). Example: 88984717
  • FC and CN must be 0–65535.
  • Invalid lines are flagged so you can correct them.

File name

Exports include today’s date so they don’t get mixed up with older files:

  • AP_Acess_Control_YYYY-MM-DD.xls
