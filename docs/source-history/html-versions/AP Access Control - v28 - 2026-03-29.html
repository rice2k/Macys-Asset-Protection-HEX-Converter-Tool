Help — What this page does

Macy's Asset Protection Control Converter converts 8-character access control values into Facility Code (FC) and Card Number (CN), supports single and batch conversion, and can import supported text and Excel-style files for faster processing.

What this page does

  • Batch convert: Convert multiple employee IDs / hex values at one time.
  • Single convert: Convert one 8-character value into FC and CN.
  • FC/CN to Hex: Build the 8-character value from a Facility Code and Card Number.
  • Import files: Import TXT, XLSX, XLSM, XML Spreadsheet, and HTML table-style Excel exports.
  • Export results: Export a formatted Excel report or TXT file using the output options you choose.

How Excel import works

The importer first looks for the strongest header match, such as Candidate Name + Colleague #. If exact headers are not found, it falls back to trusted header variations such as Colleague Name, Employee Name, Colleague ID, or Employee ID. Loose row-by-row guessing is only used as a last resort.

How to use it

  1. Batch convert: Paste your list into the main box, one value per line, then click Convert All.
  2. Import a file: Use the import button or drag and drop a supported file into the drop area.
  3. Choose export columns: Select which fields to include before exporting.
  4. Export: Use Export Excel (Formatted) or Download TXT after converting.
  5. Single / reverse convert: Use the lower sections for one-off conversions.

Input rules

  • Hex / Employee ID values must be exactly 8 characters using 0–9 and A–F.
  • FC and CN must be between 0 and 65535.
  • Full text lines such as Alan Blackmon, 88992972 are supported.
  • Invalid lines are flagged so they can be reviewed before export.

Output options

  • You can choose to include Name, Employee ID / Hex, FC, CN, Warnings / Cleanup, Input Line Number, and Converted Time.
  • If all output boxes are cleared, the export defaults back to Employee ID / Hex, FC, and CN.

Export file naming

Exports use today’s date to help keep daily runs organized:

  • AP_Acess_Control_YYYY-MM-DD.xls
  • AP_Acess_Control_YYYY-MM-DD.txt
