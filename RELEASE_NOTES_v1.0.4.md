# AP HEX Converter Tool v1.0.4

- Excel imports scan every worksheet, row, column, and cell for standalone eight-digit IDs.
- Number-only spreadsheets can be imported without name or header columns.
- Added a maximized **Pop Out Results** window with searching, status filtering, copying, horizontal scrolling, vertical scrolling, and export access.
- Long Input Queue lines wrap so their complete text remains visible.
- Updated the right banner to **AP HEX CONVERTER TOOL** and its button to **HEX TOOL**.
- Added the Salesforce Employees shortcut and clickable exported-file status links.
- Sample buttons now use working examples instead of intentional error rows.

The Windows workflow builds `AP_HEX_Converter_Tool_v1.0.4.exe` and publishes an artifact ZIP plus a SHA-256 checksum.
